# RoboDave - a powershell Module for misc stuff
[![Build Status](https://travis-ci.org/DBHeise/RoboDave.svg?branch=master)](https://travis-ci.org/DBHeise/RoboDave)



## Available cmdlets:
### Networking
* Get-LocalAddresses
* Get-LocalhostFQDN
* Get-RemoteAddresses
### Random
* Get-RandomCoordinate
* Get-RandomDateTime
* Get-RandomMEID
* Get-RandomString
### MadLib
* Use-MadLib
* New-MadLib
### Forensics
* Get-BrowserHistory
